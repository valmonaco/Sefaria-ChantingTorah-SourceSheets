Planned Features Roadmap

The Torah Chanting Source Sheet Generator is a work in progress. The current version shows its complete functionality (verse selection, trope highlighting, trope tunes, and verse chanting). In the future, multiple enhancements are planned:

  1. Customized selection of all source sheet elements: The user should be able to indicate with granularity which element to include on the sheet. For example, maybe the student wants to focus on learning a particular trope. The interface would allow selection of that trope and just present the highlighting for that trope and the tune for that trope. Another example of customization would be selecting verse printed in Hebrew without vowels.
  2. Customized selection of trope tunes and verse chanting: The current version includes only one trope cantillation system and one set of Torah verse recordings. The user should be able to select a different trope cantillation system and/or choose different recordings of the Torah verses.
  3. Account login: Currently, the user needs to create a new sheet each time he/she wants to add or remove an element. With a user registration option, the source sheet ID number could be saved in a database. After login, the user could retrieve the saved sheet and modify the elements in place on the saved sheet.
  4. Additional marginalia: During the development the Torah Chanting Source Sheet Generator, I came across interesting information about tropes and chanting (e.g., rare tropes, special chanting tunes for certain verses). I would like include some additional marginalia that will appear where appropriate.
  5. Include Haftarot and Megillot: Cantor Shermet's recordings also include verse by verse Haftarot recordings and associated Haftarah trope cantillation tunes (as well as the Megillot). Including the Haftarot and Megillot would offer additional training using the same source sheet layout.
